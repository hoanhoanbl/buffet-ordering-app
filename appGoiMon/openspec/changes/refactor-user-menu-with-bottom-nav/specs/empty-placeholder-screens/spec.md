## ADDED Requirements

### Requirement: Placeholder screens for future features
The system SHALL display placeholder screens for "Phản hồi" (Feedback) and "Liên hệ" (Contact) tabs.

#### Scenario: Feedback tab shows placeholder
- **WHEN** user navigates to the "Phản hồi" tab
- **THEN** a centered message "Tính năng Phản hồi đang phát triển" is displayed

#### Scenario: Contact tab shows placeholder
- **WHEN** user navigates to the "Liên hệ" tab
- **THEN** a centered message "Tính năng Liên hệ đang phát triển" is displayed

### Requirement: Placeholder screens have consistent styling
The system SHALL display placeholder messages in a centered, neutral style consistent with the app's theme.

#### Scenario: Placeholder text uses muted color
- **WHEN** placeholder screen is displayed
- **THEN** the text uses the app's muted brown color scheme
